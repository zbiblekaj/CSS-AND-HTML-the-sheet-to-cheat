HTML

Intro to HTML

HTML (Hyper Text Markup Language) is the skeleton of all web pages. It provides structure to the content on a website, including text, images, buttons, videos and more. CSS can then be added on top of HTML to add styling to the document.

Markup refers to a way of annotating text that is distinguishable from the text itself.

Elements

HTML annotates & separates the content within a web page using HTML tags. HTML tags are the "markup" for HTML. They are annotations that provide information to the browser about the type of context they contain.

The browser has default styling that uses the information we give it through HTML tags to design the webpage for increased readability. Of course, this design is not creative at all and doesn’t involve colors, but that’s when CSS comes in!!

A HTML ELEMENT is a unit of content in a HTML document formed by HTML tags and the text or media it contains. An example of what a HTML element looks like can be found below:

<p> Hello World! </p>

Structure

HTML is organized as a collection referred to as family tree relationships. This uses the same analogy as the family trees we see in people relationships. When an element is contained inside another element, it is referred to as the child of that element. The child element is said to be nested inside the parent of that element.

Since there can be multiple lines of NESTING, this analogy can be extended to grandchildren, great grandchildren and beyond. The relationship between elements and their ancestor and descendent elements is known as HIERACHY.

Creating a HTML document

The first step when creating a HTML file is to begin your document with a document type declaration: <!DOCTYPE html>. This should be the first line on your HTML file and will give an instruction to the browser that we are working with a HTML file. This will be on every HTML file. That specific tag refers to HTML5, as it is the current standard.

You may be wondering why the browser needs this piece of information, as it already has the file extension ‘.html’. It is crucial that you add this line of code in as although there is a chance your HTML document may work without it, you can never be sure as to how your browser will interpret your code without this.
